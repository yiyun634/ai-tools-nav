第一章：认识命令行

什么是命令行？

命令行是一个文本界面，你输入命令，电脑执行。就像和电脑说话一样，只不过要用特定的"命令"。

💡 Windows用户按 Win+R，输入 cmd 回车，打开命令提示符
Mac用户按 Command+空格，输入 terminal 回车，打开终端

常用命令（先认识一下）

1
cd 路径 - 进入某个文件夹
cd Desktop # 进入桌面 cd .. # 返回上一级 cd /home/user/doc # 进入绝对路径
2
ls 或 dir - 查看当前文件夹内容
ls # Linux/Mac dir # Windows
3
pwd - 查看当前在哪个文件夹
pwd # 显示当前路径

第二章：安装Python

为什么要装Python？

大部分AI工具都是用Python写的，安装Python后才能运行它们。

步骤1：下载Python

访问官网下载：https://www.python.org/downloads/

⚠️ 一定要下载Python 3.8或更高版本！

步骤2：安装（Windows）

  • 双击下载的安装包
  • 勾选 "Add Python to PATH"（重要！）
  • 点击 "Install Now"
  • 等待安装完成

步骤3：验证安装

打开命令行，输入：

python --version

应该显示类似 Python 3.11.0 的版本号

💡 如果显示"不是内部命令"，尝试输入 python3 --version

第三章：pip使用方法

什么是pip？

pip是Python的包管理器，相当于手机的应用商店，用来安装Python工具。

基础命令

1
安装包
pip install 包名

例如：pip install gpt4free

2
查看已安装的包
pip list
3
卸载包
pip uninstall 包名

常见问题

⚠️ 如果安装很慢或失败，使用国内镜像：
pip install 包名 -i https://pypi.tuna.tsinghua.edu.cn/simple
💡 可以把这行命令理解为"从清华大学镜像加速下载"

第四章：Git克隆项目

什么是Git？

Git是一个版本控制工具，"克隆"就是下载的意思。很多开源AI项目放在GitHub上，用Git可以下载整个项目。

步骤1：安装Git

  • Windows：https://git-scm.com 下载安装
  • Mac：打开终端，输入 git，系统会提示安装
  • Linux：sudo apt install git（Ubuntu）或 sudo yum install git（CentOS）

步骤2：克隆项目

在网站上找到项目的链接（以 https://github.com/ 开头），复制。

# 格式 git clone 复制的链接 # 示例：克隆gpt4free git clone https://github.com/xtekky/gpt4free

步骤3：使用项目

# 进入下载的文件夹 cd gpt4free # 查看里面的内容 ls # 查看安装说明（必读） cat README.md
💡 README.md 文件里详细说明了怎么使用这个项目！

第五章：实战案例

案例1：安装并使用gpt4free（免费调用GPT）

1
安装
pip install gpt4free
2
使用（创建文件 test.py）
from gpt4free import use response = use.Completion.create('gpt-3.5-turbo', '你好，介绍一下你自己') print(response)
3
运行
python test.py

案例2：安装LangChain（AI应用开发框架）

1
安装
pip install langchain
⚠️ 需要有OpenAI API Key才能使用
2
使用
from langchain.llms import OpenAI llm = OpenAI(api_key="你的API Key") result = llm("写一首关于春天的诗") print(result)

案例3：用Docker安装Dify（可视化AI工作流）

1
安装Docker

访问 https://docs.docker.com/desktop/ 下载安装

2
运行Dify
docker run -d -p 80:80 public.ecr.aws/dify/dify-community
3
打开使用

浏览器打开 http://localhost （或你的服务器IP）

❓ 常见问题

Q: 遇到"command not found"错误？

说明这个命令没找到，检查是否：

  • 命令拼写错误（注意大小写）
  • Python/Git没安装好
  • 没把路径加入系统PATH

Q: 遇到"Permission denied"错误？

权限不足，在命令前加 sudo（Linux/Mac）或以管理员身份运行（Windows）

sudo pip install 包名

Q: 不知道在哪里运行命令？

  • Windows：开始菜单 → 输入"cmd" → 回车
  • Mac：Command+空格 → 输入"terminal" → 回车
  • Linux：Ctrl+Alt+T

Q: 想放弃？不要急！

每个人都是从不会到会的。遇到问题可以：

  • 复制错误信息到搜索引擎搜索
  • 查看项目的README.md文档
  • 在GitHub项目页面提Issue问
