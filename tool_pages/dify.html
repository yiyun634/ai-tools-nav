Dify ⭐ 85k

📂 AI智能体 | 📊 入门级 | ⭐ GitHub 85k

📖 工具简介

Dify 是一个开源的 LLM 应用开发平台，提供可视化的工作流编排、RAG 引流、数据集管理等功能。生产级AI应用平台

⚡ 核心功能

  • 可视化编排：通过拖拽构建 AI 工作流
  • 多模型支持：兼容 100+ 大模型 API
  • RAG 系统：内置检索增强生成能力
  • 数据集管理：上传文档构建知识库
  • 应用模板：提供丰富的预置应用模板
  • API 部署：一键将应用发布为 API

📥 安装与使用

docker run -d -p 80:80 dify/dify

更多详细教程请访问工具官网或参考官方文档。
